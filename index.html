Description of the simulation parameters

Recovery Time = The time that it takes for the avalanche diode to recharge into its original state

Photon rate = The number of incident photons per unit time

Threshold voltage = The threshold voltage of the digital buffer connected to the avalanche diode's output


Description of the controls

Run = continuous run

Single = single step run

Join = join CH1 and CH2 traces

Split = split CH1 and CH2 traces

CH1 = Channel 1 connected to the output of avalanche diode showing an analog output

CH2 = Channel 2 connected to the output of the digital buffer showing an digital output
